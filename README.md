# is Object

Checks whether a value is an object

Because `typeof` null is a troll.

## Install

```bash
# npm
npm install --save-dev @elonehoo/is-object

# yarn
yarn add --dev @elonehoo/is-object

# pnpm
pnpm install --save-dev @elonehoo/is-object
```
